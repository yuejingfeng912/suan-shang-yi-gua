# 算上一卦 🔮

> 三法合参，白话解读。支持小六壬、梅花易数、六爻、塔罗牌。

## 在线使用

👉 **https://yuejingfeng912.github.io/suan-shang-yi-gua/**

## 使用方法

1. 用手机或电脑浏览器打开上面的链接
2. 打开后点浏览器菜单 → "添加到主屏幕"
3. 之后就可以像 APP 一样在桌面点击使用了

## 文件说明

- index.html — 完整的占卜应用（单文件，无服务器依赖）
- manifest.json — PWA 配置文件
- icons/ — 应用图标（192×192 和 512×512）

## 功能

- 🔮 小六壬快速起课
- 🌸 梅花易数随机起卦
- 🎴 六爻铜钱起卦
- 🃏 塔罗牌抽牌
- 📅 黄历查询
