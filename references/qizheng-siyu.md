# 七政四余 · 核心精要

> 七政四余是中国古代星命学的核心体系，以日月五星（七政）及四颗虚星（四余）推演人生运势。

## 四、星盘排法

### 4.1 命宫定法

1. 以出生时太阳所在宫位为基准
2. 顺数至卯时（或生时）所在宫位
3. 该宫即为命宫

**口诀：**
> 以太阳所临之宫，顺数至生时，即为命宫。

### 4.2 身宫定法

1. 以太阴（月亮）所在宫位为身宫
2. 身宫代表后天运势、配偶、身体

### 4.3 十二宫顺排

命宫确定后，十二宫按逆时针排列：
- 命宫 → 财帛 → 兄弟 → 田宅 → 男女 → 奴仆 → 夫妻 → 疾厄 → 迁移 → 官禄 → 福德 → 相貌

## 六、四余星论断

### 6.1 紫气（吉）

**性质：** 木之余气，清高福德之星

**落宫论断：**
- 命宫：气质高雅，有贵人缘
- 福德：精神富足，兴趣广泛
- 官禄：贵人提携，仕途顺遂

### 6.2 月孛（凶）

**性质：** 水之余气，暗昧情欲之星

**落宫论断：**
- 命宫：心机深沉，感情复杂
- 夫妻：婚姻多波折，桃花重
- 疾厄：女性妇科问题

### 6.3 罗睺（凶）

**性质：** 火之余气，争斗破败之星

**落宫论断：**
- 命宫：性格刚烈，易惹是非
- 财帛：破财之象，投资失利
- 官禄：官非口舌，仕途坎坷

### 6.4 计都（凶）

**性质：** 土之余气，疾病灾祸之星

**落宫论断：**
- 命宫：体弱多病，运势坎坷
- 疾厄：疾病缠身，需注意养生
- 迁移：出行不利，多阻滞

## 八、流年推断

### 8.1 流年星法

每年各星有特定行度，流年星与原盘星的关系决定该年运势：

**流年吉星照命：** 该年顺利
**流年凶星照命：** 该年阻滞
**流年星与原盘星相生：** 增益
**流年星与原盘星相克：** 损害

### 8.2 限运法

七政四余亦有限运之说，以命宫起算，逐年行限：

- 命宫限：少年运（1-15岁）
- 财帛限：青年运（16-25岁）
- 兄弟限：壮年运（26-35岁）
- ...以此类推

### 8.3 流年应事速查

## 十、实战技巧

### 10.1 快速判断命盘层次

1. 看命宫主星：吉星为佳
2. 看太阳位置：庙旺为佳
3. 看太阴位置：庙旺为佳
4. 看四余分布：凶星少为佳

### 10.2 事业判断

- 官禄宫有吉星：事业顺利
- 太阳庙旺：有贵气
- 木星入官禄：学业仕途
- 金星入官禄：艺术商业

### 10.3 财运判断

- 财帛宫有吉星：财运佳
- 太阴庙旺：偏财运
- 金星入财帛：正财偏财皆佳
- 木星入财帛：投资运

### 10.4 感情判断

- 夫妻宫有吉星：婚姻美满
- 太阴庙旺：配偶贤惠
- 金星入夫妻：感情顺利
- 月孛入夫妻：桃花多
