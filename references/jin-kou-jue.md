# 神课金口诀 · 核心精要

> 金口诀是中国古代预测学三大式之一，以天干地支配合神煞、贵神断事，准确快速，被称为"袖中金"。

---

## 一、金口诀基础

### 1.1 金口诀特点

- **快速**：起课简单，断事迅速
- **准确**：古有"金口一开，百无失一"之说
- **实用**：侧重人事占问，直接切入核心

### 1.2 四课构成

金口诀以**四课**断事，自上而下：

| 位置 | 名称 | 主事 |
|------|------|------|
| 人元 | 天干 | 外应、来意、表象 |
| 贵神 | 地支（将神） | 贵人、贵贱、层次 |
| 将神 | 地支 | 动态、事情发展 |
| 地分 | 地支 | 结果、定局、归宿 |

### 1.3 排课规则

**人元取法：** 随贵神、将神、地分而定
**贵神取法：** 以日干、时辰推算
**将神取法：** 以月将加时推算
**地分取法：** 以问事方位或数字推算

## 四、地分取法

### 4.1 方位取地分

| 方位 | 地支 |
|------|------|
| 北 | 子 |
| 东北 | 丑、寅 |
| 东 | 卯 |
| 东南 | 辰、巳 |
| 南 | 午 |
| 西南 | 未、申 |
| 西 | 酉 |
| 西北 | 戌、亥 |

### 4.2 数字取地分

以问事时的数字推算：
- 1 → 子
- 2 → 丑
- 3 → 寅
- ... 以此类推，12 → 亥，超过12则除以12取余数

### 4.3 时辰取地分

以问事时辰的地支为地分。

## 六、断课方法

### 6.1 主客论

- **人元**：为客、为外、为主动方
- **地分**：为主、为内、为被动方

**客克主：** 主动方有利
**主克客：** 被动方有利
**主客相生：** 和谐顺利
**主客比和：** 势均力敌

### 6.2 生克论

| 关系 | 论断 |
|------|------|
| 上生下 | 贵人相助，顺利 |
| 下生上 | 自力更生，辛苦 |
| 上克下 | 阻力较大，不利 |
| 下克上 | 反抗有力，可成 |
| 上下比和 | 势均力敌，平稳 |

### 6.3 贵神将神断

**贵神断：**
- 贵神为吉神：事情顺利
- 贵神为凶神：事情阻滞
- 贵神空亡：事情落空

**将神断：**
- 将神生贵神：事情顺利
- 将神克贵神：事情反复
- 将神与贵神比和：平稳

### 6.4 四课综合断

```
人元（外应）→ 来意、表象
    ↓
贵神（贵人）→ 贵贱、层次
    ↓
将神（动态）→ 发展、变化
    ↓
地分（定局）→ 结果、归宿
```

### 9.5 胜光（午）

**主事：** 传送、消息、联络

### 9.10 登明（亥）

**主事：** 智慧、谋略、隐秘
**论断：** 登明临旺，谋略深远；登明受克，计谋败露。

**主事：** 隐秘、阴私、女性

**主事：** 稳重、积累、田宅
**论断：** 大吉临旺，积累有成；大吉受克，根基动摇。

## 十、实战案例

### 10.1 求财案例

**问：** 今日求财如何？

**课式：**
```
人元：甲
贵神：青龙（寅）
将神：从魁（酉）
地分：丑
```

**分析：**
- 青龙为财喜之神，从魁为财帛将神
- 寅木克丑土，上克下，主动求财
- 贵神青龙吉，将神从魁吉
- 地分丑土受克，求财辛苦

**断：** 可求财，但需主动出击，辛苦求财。

### 10.2 婚姻案例

**问：** 婚姻可成否？

**课式：**
```
人元：丙
贵神：六合（卯）
将神：小吉（未）
地分：亥
```

**分析：**
- 六合为婚姻和合之神
- 卯木与亥水相生，下生上
- 将神小吉主小喜
- 地分亥为登明，主智慧谋略

**断：** 婚姻可成，双方有感情基础。

### 10.3 官讼案例

**问：** 官司胜负？

**课式：**
```
人元：庚
贵神：勾陈（辰）
将神：天罡（辰）
地分：申
```

**分析：**
- 勾陈、天罡皆为刑狱之神
- 辰土生申金，下受上生
- 人元庚金克贵神辰土，客克主

**断：** 官司不利，主方败诉。
